call, apply, and bind

In this assignment, we will explore the concepts of call, apply, and bind in JavaScript.

call

The call() method allows you to call a function with a specified this keyword, value and arguments provided individually.
example: 
function greet(greeting) {
            return `${greeting}, ${this.name}`;
          }

          const person = { name: 'Alice' };
          console.log(greet.call(person, 'Hello')); // Output: Hello, Alice

apply

The apply() method is similar to call(), but it takes an array of arguments instead of individual arguments.
example: 
function greet(greeting) {
            return `${greeting}, ${this.name}`;
          }

          const person = { name: 'Alice' };
          console.log(greet.apply(person, ['Hello'])); // Output: Hello, Alice

bind

The bind() method creates a new function that, when called, has its this keyword set to the provided value, with a given sequence of arguments preceding any provided when the new function is called.
example: 
function greet(greeting) {
            return `${greeting}, ${this.name}`;
          }

          const person = { name: 'Alice' };
          const greetAlice = greet.bind(person);
          console.log(greetAlice('Hello')); // Output: Hello, Alice
